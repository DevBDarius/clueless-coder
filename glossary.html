Glossary

Welcome to Clueless Coder's Glossary of Computer and Programming Terminology. The definitions listed here are opinionated and in no way exhaustive, but the list of available definitions will expand as we become a little less clueless! 😃

A

absolute path
The name of a file and the full list of directories that contain the file. In a URL, this starts at the site's URL or IP; in a file system, it starts at the root directory.

B

Boolean
In the context of programming, a Boolean is generally a data type associated with logic, which can either be True or False. The word comes from the name of George Boole, the creator of a logic-based form of algebra that also bears his name.

C

Cascading Style Sheets (CSS)
A language used to tell a browser how to display the content found in an HTML document.
Clueless
Colloquially, a term to describe a person who does not understand something and is unaware of his/her ignorance. Here, it describes a person who does not assume knowledge and is willing to admit that there is and always will be more that he/she can learn.

D

duck typing
A feature of certain dynamic programming languages that allows the interpreter to ascertain the type of an object based solely on the object's appearance. The term comes from the expression "If it looks like a duck and quacks like a duck, then it is a duck."
