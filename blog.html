Where to Start?

Luckily for anyone trying to learn about computers and programming, there are so many free resources available online. Gone are the days of flipping through dry, technical manuals.

I'm going to spend the first few weeks of my exploration working through the exercises offered in these popular computer programming web sites:

  • http://www.codecademy.com/

  • http://www.codeavengers.com/

  • http://www.khanacademy.org/computing/cs

At this early stage, I'm still trying to figure out what languages are the most useful/accessible/widely used.  But, I do know that the web is built on HTML and JavaScript, so these seem like good places to start.

If anyone knows of other sites like these or has recommendations, please add a comment below and I'll check them out.
